安装方法:

1.产看node版本，没安装的请先安装；

 $  node -v

2.清楚node缓存；

$  sudo npm cache clean -f  

3.安装node版本管理工具'n';

$  sudo npm install n -g

4.使用版本管理工具安装指定node或者升级到最新node版本；

$  sudo n stable  （安装node最新版本）

$  sudo n 8.9.4 （安装node指定版本8.9.4）

5.使用node -v查看node版本，如果版本号改变为你想要的则升级成功。


npm  Ubuntu  升级方法
sudo npm cache clean -f
sudo npm install -g n
sudo n stable
